SQL Challenge


SELECT mov_title AS "Movie", 
mov_year AS "Release Year", 
genres.gen_title AS "Genre", 
director.dir_fname || ' ' || director.dir_lname AS "Director"

FROM movie

JOIN rating ON rating.mov_id = movie.mov_id
JOIN movie_genres ON movie_genres.mov_id = movie.mov_id
JOIN genres ON genres.gen_id = movie_genres.gen_id
JOIN movie_direction ON movie_direction.mov_id=movie.mov_id
JOIN director ON director.dir_id=movie_direction.dir_id

WHERE director.dir_fname = 'Danny'

AND movie.mov_time < (
    SELECT mov_time FROM movie
    WHERE movie.mov_title = 'Slumdog Millionaire'
)

The objective of this query was both to challenge myself and consolidate the knowledge I gained after learning SQL.

The query sorts through a movie database looking for films directed by Danny Boyle that are shorter in runtime than Slumdog Millionare.

It outputs the film title, release yeasr, genre and director of all films that meet those conditions in that database.
